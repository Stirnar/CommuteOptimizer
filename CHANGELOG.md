V2.2 - January 2026

New Features
- Track Comparison Mode: Enter your home address and see which tracks have the lowest commute burden
- Two-way optimization: Find optimal home for a track OR find optimal track for your home
- Top 5 Rankings: See best tracks ranked
- Detailed cost breakdowns: Gas, maintenance, and opportunity costs for each track
- Drill-down capability: View full details for any ranked track

Improvements
- Fixed map display: Map tiles now load properly (no more gray squares)
- Progress indicators: Shows calculation progress during long operations
- Mode toggle: Easy switching between optimization modes
- Better loading messages: Clear feedback on what's happening

What You Can Do Now
- Enter "Plantation, FL" and find your 5 best tracks
- Compare tracks based on YOUR actual home location
- See exactly how much time and money each track costs
- Make informed track selection decisions

 Performance Notes
- Track comparison takes 3-5 minutes (analyzing all 60 tracks)
- Single track calculation: 5-30 seconds depending on whether optimal location search is needed

	-

V2.1 - January 2026

New Features
- Address input: Enter street addresses instead of coordinates
- Interactive maps: Visual display of home, clinical sites, and NSU campus
- Geocoding: Automatic address-to-coordinate conversion
- Smart detection: Automatically recognizes address vs coordinate format

Improvements
- External data loading: CSV files loaded separately for easier updates
- Better error messages: Clear feedback on invalid addresses
- South Florida validation: Ensures addresses are in service region
- Privacy focused: All calculations client-side

What You Can Do Now
- Type "123 Main St, Fort Lauderdale, FL" instead of coordinates
- See your commute pattern on an interactive map
- Click markers to see site details
- Pan and zoom the map

	-

V2.0 - January 2026

Initial Features
- Track selection: Choose from 60 available tracks
- Commute calculation: Calculate time and costs for rotations
- Optimal location finder: Find best home location for selected track
- Cost analysis: Gas, maintenance, and opportunity costs
- Block breakdown: Detailed view of each rotation block

Key Metrics
- Hours per day/week/month/year
- Total miles driven
- UWorld questions lost to commuting
- Financial opportunity cost
