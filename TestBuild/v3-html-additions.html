üè† Getting Your Home Coordinates

  1. Open Google Maps
  2. Search for your home address or right-click anywhere on the map
  3. Click on the coordinates that appear at the top or in the context menu
  4. The coordinates will be copied to your clipboard (format: latitude, longitude)
  5. Paste the coordinates into the "Home Location" field
Example: 26.082, -80.249

üéØ Using the Custom Block Builder

  1. Click the "Build Custom Block" button on the main page
  2. Enter a name for your custom block (e.g., "My Elective Rotation")
  3. Add weeks to your block schedule - each week can have different locations
  4. For each week, specify which days you'll be at which locations
  5. You can use existing locations from the dropdown or add custom locations
  6. To add a custom location: enter the name and coordinates (lat, long)
  7. Choose if your block has Wednesday campus trips (most blocks do)
  8. Save your custom block - it will be added to your track list

üó∫Ô∏è Adding Custom Locations

If you need to add a rotation site that's not in our database:

  1. Get the coordinates from Google Maps (see above)
  2. In the Custom Block Builder, find the "Custom Locations" section
  3. Enter the location name (e.g., "Private Practice - Dr. Smith")
  4. Enter the coordinates in the format: latitude, longitude
  5. Click "Add Location" to save it to your custom locations list
  6. The location will now appear in the location dropdowns

üí° Tips

  • Use the optimal location finder if you're flexible about where to live
  • Custom blocks can have complex schedules (e.g., different sites on different days)
  • Wednesday campus trips add an extra commute leg to account for lecture days
  • All calculations use real driving routes from OpenStreetMap
  • You can adjust cost parameters (gas price, MPG, salary) in the settings
