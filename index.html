Calculation Methodology

  • Address geocoding: Nominatim (OpenStreetMap) with 1-second rate limiting
  • Interactive map: Leaflet.js with OpenStreetMap tiles
  • Routes calculated using OpenStreetMap OSRM
  • Wednesday campus trips: Home ‚Üí Site ‚Üí School ‚Üí Home (3 legs)
  • Regular days: Home ‚Üí Site ‚Üí Home (2 legs)
  • Blocks at Nemours = 0 commute (on-site housing)
  • Car maintenance cost: $0.0986/mile (AAA average)
  • UWorld questions: 1.5 minutes per question max
  • Opportunity cost: Resident hourly rate = Annual salary √∑ (52 weeks √ó 60 hrs/week)
  • Optimal location: If no home location provided, system tests 500 points across South Florida to find location that minimizes total commute hours

Data Source & Privacy

Privacy: All calculations are performed in your browser. Address geocoding uses OpenStreetMap's free API. Your location data is never stored or transmitted to any server beyond the geocoding lookup.
