Calculation Methodology

  • Routes calculated using OpenStreetMap OSRM
  • Wednesday campus trips: Home → Site → School → Home (3 legs)
  • Regular days: Home → Site → Home (2 legs)
  • Blocks at Nemours = 0 commute (on-site housing)
  • Car maintenance cost: $0.0986/mile (AAA average)
  • UWorld questions: 1.5 minutes per question average
  • Opportunity cost: Resident hourly rate = Annual salary ÷ (52 weeks × 60 hrs/week)
  • Optimal location: If no home location provided, system tests 500 points across South Florida to find location that minimizes total commute hours
